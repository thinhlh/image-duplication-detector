# Image & Video duplicator detection

# Purpose
Image & Video duplicator detection is a CLI tool written in golang, that will browse a specfic macos folder (including videos & photos). With a given similarity percentage (e.g 90% or 95%). It will output a list of items that are identified as similar.

# Techstack
This application is a CLI tool written in golang, the code structure should be user-friendly, primary features includes:
- Go CLI to ask for the folder to scan
- The similarity percentage, if >= the given percentage, it will output files that has similarity
- The output should be readable, traceable & easy to open from the CLI.
